## 简介
本仓库放的是Windows(64位)平台SublimeText3官方下载后，安装一些插件的版本,安装过程见博客[http://blog.csdn.net/yanzi1225627/article/details/54881235](http://blog.csdn.net/yanzi1225627/article/details/54881235).
